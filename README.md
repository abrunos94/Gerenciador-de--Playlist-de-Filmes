Seja bem-vindo ao **MyMovies**! Este projeto é um gerenciador de listas de filmes que permite pesquisar títulos em tempo real e organizar uma playlist personalizada. Desenvolvido como parte dos meus estudos na **Hashtag Treinamentos**, ele demonstra a aplicação prática de conceitos fundamentais de desenvolvimento web.

## 🛠️ Tecnologias e Recursos Utilizados

Para a construção desta aplicação, foram aplicados os seguintes conhecimentos:

* **Manipulação de DOM:** Controle total dos elementos HTML via JavaScript para criar uma experiência dinâmica.
* **Consumo de API:** Integração com a [OMDB API](https://www.omdbapi.com/) utilizando `Fetch` para busca de dados reais de filmes.
* **Persistência de Dados:** Uso de `LocalStorage` para garantir que sua lista de filmes não seja perdida ao atualizar a página.
* **Estilização Avançada:** CSS3 com foco em responsividade e variáveis para manutenção de cores.
* **Bibliotecas Externas:** * [Notie.js](https://github.com/jaredreich/notie) para notificações interativas e modais de confirmação.
    * [Bootstrap Icons](https://icons.getbootstrap.com/) para o enriquecimento visual da interface.

## 🔑 Sobre a Chave de API (OMDB)

Neste repositório, optei por deixar o arquivo de configuração da chave de API (`key.js`) visível e acessível. 

**O motivo:** Fiz isso propositalmente para facilitar o teste imediato da aplicação por recrutadores e outros desenvolvedores. Dessa forma, o projeto funciona "out of the box" no GitHub Pages, sem a necessidade de configurações manuais de variáveis de ambiente por parte de quem está avaliando o código.
