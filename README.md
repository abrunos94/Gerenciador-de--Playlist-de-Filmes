Seja bem-vindo ao **MyMovies**! Este é um projeto prático que desenvolvi durante meus estudos na **Hashtag Treinamentos**, focado em criar uma experiência interativa para buscar e organizar listas de filmes personalizados.

---

## 🚀 Funcionalidades Interativas

- **Busca Dinâmica:** Pesquise filmes em tempo real integrando com a API do OMDB.
- **Filtro por Ano:** Refine sua busca informando o ano de lançamento.
- **Gestão de Playlist:** Adicione seus filmes favoritos à lista e remova-os com um clique.
- **Feedback Visual:** Sistema de notificações e confirmações estilizadas (via biblioteca Notie.js).
- **Interface Responsiva:** Design limpo e adaptável para diferentes tamanhos de tela.

---

## 🛠️ Tecnologias Utilizadas

Para construir este gerenciador, utilizei:

- **Lógica de Programação:** JavaScript (ES6+) para manipulação de arrays e objetos.
- **Integração:** Fetch API para consumo de dados da [OMDB API](https://www.omdbapi.com/).
- **Interface:** HTML5 semântico e CSS3 com variáveis para padronização de cores.
- **Bibliotecas:** - [Bootstrap Icons](https://icons.getbootstrap.com/) para os elementos visuais.
  - [Notie.js](https://github.com/jaredreich/notie) para alertas e modais de confirmação.
